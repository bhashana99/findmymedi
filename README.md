# FindMyMedi (Ongoing)

FindMyMedi is a healthcare platform designed to provide real-time medicine availability tracking and secure pharmacy integration.  
The system aims to improve accessibility to essential medicines by connecting users with pharmacies through a scalable and secure architecture.

---

## Project Overview

FindMyMedi is being developed as an enterprise-level healthcare solution with a focus on:

- Real-time medicine availability updates  
- Secure authentication and access control  
- Scalable backend services for pharmacy and user management  

This project demonstrates practical experience in building full-stack applications with microservices and modern authentication standards.

---

## Key Features

- Real-time medicine availability tracking  
- Secure pharmacy integration and management  
- REST API-based communication between services  
- OAuth2/OpenID Connect authentication via WSO2  
- Containerized deployment support using Docker  

---

## Tech Stack

- **Backend:** Java, Spring Boot, Microservices  
- **Frontend:** Angular  
- **Database:** PostgreSQL  
- **Authentication:** WSO2, OAuth2, OpenID Connect  
- **Communication:** gRPC (planned/ongoing)  
- **Deployment:** Docker
